**Title:**
SQLMap Basics – Automated SQL Injection Testing

**Description:**
Practiced the fundamentals of SQLMap, a powerful open-source tool used for detecting and exploiting SQL Injection 
vulnerabilities in web applications. During this hands-on exercise, I explored how SQLMap can identify vulnerable 
parameters, enumerate databases, retrieve database information, and automate many tasks that would otherwise be performed
manually. This practice helped me better understand the impact of SQL Injection vulnerabilities and the importance of secure 
input validation and database security. All testing was conducted in a controlled lab environment for educational and ethical
learning purposes.
