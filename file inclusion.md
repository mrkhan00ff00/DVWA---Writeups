
**Title:**
File Inclusion Vulnerability – Hands-On Lab Practice

**Description:**
Completed a hands-on lab focused on understanding File Inclusion vulnerabilities in web applications. During this exercise,
I explored how improper handling of file input parameters can allow attackers to include and access unintended files on a server. 
By working through different scenarios in a controlled lab environment, I gained practical insight into the risks associated with 
Local File Inclusion (LFI) and Remote File Inclusion (RFI), including unauthorized file access, information disclosure, and potential 
system compromise. This lab helped me better understand secure coding practices, input validation, and the importance of properly
handling user-supplied data to prevent file inclusion attacks.
