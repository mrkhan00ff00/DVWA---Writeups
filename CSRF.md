**Title:**
Understanding CSRF (Cross-Site Request Forgery) Through Hands-On Lab Practice

**Description:**
Completed a hands-on lab focused on understanding Cross-Site Request Forgery (CSRF), one of the common web application 
security vulnerabilities. During this exercise, I explored how attackers can trick authenticated users into performing 
unintended actions on a web application without their knowledge. By testing different security levels in a controlled lab 
environment, I gained practical insight into how CSRF attacks work, their potential impact on user accounts and application 
security, and the importance of implementing proper defenses such as CSRF tokens, secure session management, and user verification
mechanisms. This lab helped strengthen my understanding of web application security and secure development practices.
